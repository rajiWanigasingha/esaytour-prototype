## Installation Guide

To review the working prototype of the Easy Tour mobile application, an Android APK file has been provided.

Please follow the steps below to install the application on your Android device:

1. Download the provided APK file to your mobile device.
2. Open the downloaded file.
3. If prompted, enable **Install from Unknown Sources**  
   (this may appear as a security setting depending on your device).
4. Proceed with the installation.
5. Open the application once installation is complete.
